# Star Wars Project 🚀

Este é um projeto que consome a API SWAPI para exibir personagens, planetas e recomendações de filmes de Star Wars.

## 🛠 Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  
- API SWAPI (https://swapi.dev/)

## 📂 Estrutura do Projeto
